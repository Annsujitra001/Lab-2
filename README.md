# Lab-2

1. เลือกพื้นที่ศึกษาและเหตุผล

       พื้นที่ศึกษา: จังหวัดเชียงใหม่
       เพราะ จังหวัดเชียงใหม่ถูกเลือกเป็นพื้นที่ศึกษา เนื่องจากมีความหลากหลายทางภูมิประเทศ ทั้งพื้นที่ภูเขา ป่าไม้ และพื้นที่เกษตรกรรม อีกทั้งยังเผชิญกับปัญหาฤดูแล้งและไฟป่าซึ่งส่งผลต่อความหนาแน่นของพืชพรรณและความชื้นในพื้นที่

2. เลือกดาวเทียมและช่วงเวลา

       2.1 เลือกใช้ Sentinel-2 เพราะ
              > มีความละเอียดเชิงพื้นที่สูง (10 เมตร)
              > เหมาะสำหรับการวิเคราะห์ vegetation และ water indices
              > มี revisit time สั้น ทำให้ได้ข้อมูลต่อเนื่อง
       2.2 เลือกช่วงเวลานี้ เพราะ
              > มกราคม 2023 (ก่อนฤดูแล้ง)
              > เมษายน–พฤษภาคม 2023 (ช่วงฤดูแล้ง)
	   2.3 Cloud Filter
              > กำหนด < 20% เพื่อลดผลกระทบจากเมฆ

Methodology

       1. Data Preparation
              > เลือกพื้นที่จังหวัดเชียงใหม่จาก FAO GAUL dataset
              > กรองภาพ Sentinel-2 ตามช่วงเวลาและ cloud percentage
              > เลือกเฉพาะ band ที่จำเป็น (B2, B3, B4, B8) เพื่อแก้ปัญหา band ไม่สม่ำเสมอ
              > สร้าง composite ด้วย median และ clip ตามขอบเขตพื้นที่ศึกษา
	   2. Spectral Indices
		      2.1 NDVI (Normalized Difference Vegetation Index) 
                  ใช้วัดความหนาแน่นของพืชพรรณ
			                สูตร: NDVI = (B8 - B4) / (B8 + B4)
		      2.2 NDWI (Normalized Difference Water Index) 
                  ใช้วัดความชื้นและแหล่งน้ำ
			                สูตร: NDWI = (B3 - B8) / (B3 + B8)
        3. Temporal Analysis (Comparison)
		      มีการเปรียบเทียบ NDVI ระหว่าง:
                > มกราคม (ก่อนแล้ง)
                > เมษายน (ช่วงแล้ง)
	          Result: NDVI ลดลงในหลายพื้นที่ โดยเฉพาะพื้นที่เกษตรกรรม
		      Interpretation: แสดงถึงผลกระทบของฤดูแล้งที่ทำให้พืชพรรณลดความหนาแน่นลง
	    4. NDVI Change Map
		      คำนวณ: NDVI Change = NDVI (April) - NDVI (January)
		      Interpretation:
                > สีแดง --> พืชลดลง
                > สีเขียว --> พืชเพิ่มขึ้น
                > สีขาว --> ไม่เปลี่ยน
		      Result: พื้นที่ส่วนใหญ่แสดงสีแดง แสดงถึงการลดลงของพืชพรรณในช่วงฤดูแล้ง
		5. Visualization
			  มีการแสดงผลแผนที่:
			  	> NDVI (Jan 2023)
				> NDVI (Apr 2023)
				> NDVI Change
				> NDWI
			  พร้อมการใช้ color scale เพื่อช่วยในการตีความข้อมูล

3. วิเคราะห์และตีความ (ต้องทำอย่างน้อย 3 ข้อจาก 5)
	
		3.1 คำนวณและแสดงผล Spectral Index ที่เหมาะสมกับพื้นที่ศึกษา (ไม่จำกัดเฉพาะ NDVI)
			NDVI
   				> ค่าสูง --> พืชหนาแน่น (พื้นที่ป่า)
   				> ค่าต่ำ --> พื้นที่เมือง/ดินเปล่า
			NDWI
   				> ค่าสูง --> แหล่งน้ำ
   				> ค่าต่ำ --> พื้นที่แห้ง
		3.2 Zonal Statistics
			อำเภอที่มีค่า NDVI สูงสะท้อนพื้นที่ป่าหรือเกษตรกรรมหนาแน่น ในขณะที่อำเภอเมืองมีค่า NDVI ต่ำเนื่องจากพื้นที่สิ่งปลูกสร้าง
		3.3 วิเคราะห์ความสัมพันธ์ระหว่าง Index 2 ตัว และ Interpret ผล
   				> NDVI สูง → NDWI มักต่ำ (พื้นที่พืชหนาแน่น = น้ำผิวดินน้อย)
   				> NDVI ต่ำ → NDWI อาจสูง (เมือง/พื้นที่น้ำ/พื้นที่โล่ง)
			NDVI และ NDWI มีความสัมพันธ์แบบผกผันในหลายพื้นที่ เนื่องจากพื้นที่พืชพรรณหนาแน่นจะลดสัดส่วนของน้ำผิวดิน

4. Data Export

   		> Format: GeoTIFF
   		> CRS: EPSG:4326 (มาตรฐานทั่วไป ใช้งานง่าย)
   		> Scale: 30 เมตร (ลดขนาดไฟล์และยังคงความละเอียดที่เหมาะสม)
